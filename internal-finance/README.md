# internal-finance starter repo layout

This folder contains the requested project setup:

- `frontend/` → Next.js + TypeScript + Tailwind + dark mode starter.
- `template-.NET10-domain/` → .NET domain template content to be used as the backend template base.

## Notes
- The .NET template files are copied from the existing template baseline in this workspace.
- `template-.NET10-domain/README.md` provides a .NET 10-focused entry README.
