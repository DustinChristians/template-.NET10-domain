# Frontend (Next.js)

This is an Option 1 starter: **Next.js + TypeScript + Tailwind + shadcn/ui patterns + dark mode**.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Copy `.env.example` to `.env.local` and set your backend API URL:

```bash
cp .env.example .env.local
```

## Backend integration

Use `NEXT_PUBLIC_API_BASE_URL` in your API client to call your .NET backend.
